# 📦 QR Code Generator (Python)

A simple and beginner-friendly **QR Code Generator** built using Python.  
This project allows users to convert any text or URL into a QR code image.

---

## 🚀 Features

- 🔗 Generate QR code from text or URL
- 🖼️ Save QR code as image file (.png)
- ⚡ Fast and lightweight script
- 👶 Beginner-friendly code

---

## 🛠️ Technologies Used

- Python
- qrcode library
- Pillow (PIL)

---

## 📦 Installation

### 1. Clone the repository
```bash
git clone https://github.com/your-username/qr-code-generator-python.git
cd qr-code-generator-python
2. Install dependencies
pip install qrcode[pil]

▶️ How to Run

python app.py

🧠 How It Works

1.User enters text or link
2.Python converts it into QR code
3.QR code is generated using qrcode library
4.Image is saved as .png file
